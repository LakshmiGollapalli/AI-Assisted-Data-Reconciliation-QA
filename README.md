# AI-Assisted Data Reconciliation & QA Project

## Overview
This project demonstrates end-to-end data testing and reconciliation using sample enterprise data. 
It includes requirement analysis, data validation, reconciliation testing, defect reporting, and traceability.

## Key Skills Demonstrated
- Data Testing & Reconciliation
- SQL / PL-SQL Validation
- Excel-based Test Cases & Bug Reports
- Python (Pandas) for Data Analysis
- API Testing (Postman)
- Unix/Linux basics
- AI-assisted data generation and analysis

## Note
This is a self-initiated practice project based on sample requirements, created for learning and demonstration purposes.
